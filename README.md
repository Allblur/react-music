# vue-musicbox

A musicbox app build with react and redux.

## Build Setup

```bash
npm install
```

## run dev server

```bash
npm start
```

## run UT

```bash
npm test
```

## See UT coverage

```bash
npm run test:coverage
```

## build package

```bash
npm run build
```

## More

git checkout real-world
